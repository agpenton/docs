## Overview

Welcome to **Asdrubal's Corner** — a personal knowledge base documenting real-world DevOps engineering experience.

This site is built around practical knowledge: patterns that work in production, tools worth knowing, and concepts explained clearly. If you work in DevOps, cloud infrastructure, or platform engineering, you should find something useful here.

## What's Inside

### DevOps

The core section. Covers the tools, practices, and mindset that make software delivery fast and reliable.

- **CI/CD** — GitHub Actions, GitLab CI, Jenkins, and pipeline patterns that ship code safely.
- **Infrastructure as Code** — Terraform, Ansible, and Pulumi for repeatable, version-controlled infrastructure.
- **Monitoring & Observability** — Prometheus, Grafana, and OpenTelemetry for production confidence.
- **Virtualization & Containers** — Kubernetes, Docker, and cloud-native workload patterns.
- **Programming** — Scripts, tooling, and code that supports DevOps workflows.

### Topics

A curated index of tools and articles worth reading, organized by theme.
